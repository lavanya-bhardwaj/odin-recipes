# Odin Recipes

A basic recipe website built as part of The Odin Project Foundations course.

## Project Overview
This project consists of a main index page containing links to multiple recipe pages. The goal of this project is to practice fundamental HTML concepts and build a strong foundation in structuring web pages.

## Skills Demonstrated
- Creating and structuring HTML documents
- Working with headings, paragraphs, lists, and links
- Organizing files and folders
- Using Git and GitHub for version control
- Building a multi-page website

## Future Improvements
This project will later be revisited and styled using CSS to improve its appearance and layout.
